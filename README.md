recipebook
==========

I'm writing a cookbook to learn Python, Postgresql, and other potential tools.

# Prerequisites

Install the following software.

|              pyenv             |           poetry           |
|:------------------------------:|:--------------------------:|
| https://github.com/pyenv/pyenv | https://python-poetry.org/ |


# Build

```
poetry install
poetry run example
```

# Package

```
poetry build
```

# Deploy

```
poetry publish
```
